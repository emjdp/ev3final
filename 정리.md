# EV3 새 프로젝트 필수값 정리

## 핀맵

| 역할 | 포트 | 메모 |
|---|---|---|
| 왼쪽 주행 모터 | `outA` | `LargeMotor` |
| 오른쪽 주행 모터 | `outB` | `LargeMotor` |
| 그리퍼/리프트 모터 | `outC` | `MediumMotor` |
| 왼쪽 컬러센서 | `in1` | 좌측 라인/노드 감지 |
| 중앙 컬러센서 | `in2` | 라인/색상 마커 감지 |
| 오른쪽 컬러센서 | `in3` | 우측 라인/노드 감지 |
| 초음파 센서 | `in4` | 물체 거리 |

주행 속도 부호: `left_speed = base_speed - turn`, `right_speed = base_speed + turn`.

| 노랑 | `4` |
| 빨강 | `5` |
| 흰색 | `6` |

## 반사광 threshold

검정 판정 규칙: `raw < threshold` 이면 검정(`1`), 아니면 흰색/바닥(`0`).    

| 센서 | threshold |
|---|---:|
| 왼쪽 `in1` | `43` |
| 중앙 `in2` | `36` |
| 오른쪽 `in3` | `42` |

### 최신 통합 실험값

중앙 센서는 색상모드로 `COLOR_BLACK(1)` 여부를 보고, 좌/우만 반사광 threshold를 쓴다.

| 용도 | 왼쪽 | 오른쪽 |
|---|---:|---:|
| 조향용 `*_th_steer` | `66` | `63` |
| 노드용 `*_th_node` | `18` | `14` |

## 엔코더/기하값

| 값 | 숫자 |
|---|---:|
| 90도 회전 기본 엔코더각 `BASE_PIVOT_DEG_90` | `193.0` |
| 180도 회전 기본 엔코더각 `BASE_PIVOT_DEG_180` | `386.0` |
| 바퀴 지름 가정 `WHEEL_DIAM_MM` | `56.0` |
| 1도당 이동거리 `MM_PER_DEG` | 약 `0.4887 mm/deg` |
| 회전 후 대기 `post_turn_settle_ms` | `90` 또는 Stage 2 기준 `120` |

## 동작 메모

- BACK 버튼은 프로그램 입력으로 쓰지 않는다. 정지는 `robotctl stop`, Ctrl-C, 또는 별도 안전 입력.
